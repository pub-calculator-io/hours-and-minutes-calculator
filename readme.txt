This online hours calculator is an easy way to count hours and minutes at the touch of a button.

[https://www.calculator.io/hours-and-minutes-calculator/](https://www.calculator.io/hours-and-minutes-calculator/)

== Usage ==

Add the Hours and Minutes Calculator shortcode to your page, post or sidebar:

`[ci_hours_and_minutes_calculator]`

Add the following code to your template where you would like the Hours and Minutes Calculator to appear:

`<?php display_ci_hours_and_minutes_calculator(); ?>`

== Installation ==

1. Upload the Hours and Minutes Calculator /ci_hours_and_minutes_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Hours and Minutes Calculator plugin through the "Plugins" menu in WordPress.
